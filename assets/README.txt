Drop mood-board images, 3D models & ORB VIDEOS here.

Orb accretion videos (play on hover, warped into the swirl):
  assets/<stageid>.mp4   e.g. boot.mp4, feed.mp4, grow.mp4, corrupt.mp4, glitch.mp4
  (mp4/webm, muted, ideally looping; missing files fall back to fluid-only)

Soundtracks (per act, shown as the waveform scrubber in the Timeline play bar):
  assets/<stageid>.mp3   (or .ogg/.webm/.wav)  e.g. init.mp3, glitch.mp3
  Missing files fall back to a generated royalty-free loop. Use only audio you
  have the right to use — do NOT drop ripped/copyrighted tracks here.

Mood images: <stageid>-1.jpg ... e.g. init-1.jpg, overfit-2.jpg
Stage ids (the cycle): init, grow, optimize, overfit, glitch
Models: any .obj/.fbx, then list them in src/config/stages.js MODELS[]

Place geometry / panoramas live in assets/materials/ (used by the Layout view).
Raw screen recordings live in assets/records/ (gitignored).
